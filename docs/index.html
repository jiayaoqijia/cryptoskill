Get Started

Works with every agent

Install skills for Claude Code, OpenClaw, Codex, and more.

# Add to your project's Claude Code skills
$ git clone https://github.com/jiayaoqijia/cryptoskill.git /tmp/cs
$ cp -r /tmp/cs/skills/exchanges/binance-spot-api .claude/skills/
Clones the registry and copies a skill into your project's .claude/skills/ directory for Claude Code to use.
# Add a hosted MCP server to Claude Code
$ claude mcp add blockscout https://mcp.blockscout.com/mcp
# Or install from GitHub and run locally via npx
$ npx @solana/mcp-server
# Or clone and build from source
$ git clone https://github.com/solana-foundation/solana-mcp-official.git
$ cd solana-mcp-official && npm install && npm start
Use claude mcp add for hosted MCP servers, npx for npm packages, or clone from GitHub for full control.
# Install via ClawHub CLI
$ npm i -g clawhub
$ clawhub install binance-spot-api
ClawHub CLI installs skills directly into your OpenClaw workspace with one command.
# Copy to any agent's workspace skills directory
$ git clone https://github.com/jiayaoqijia/cryptoskill.git
$ cp -r cryptoskill/skills/exchanges/binance-spot-api ~/workspace/skills/
Clone the full registry and manually copy any skill folder into your agent's skills directory.

Compatible with Claude Code, OpenClaw, Codex, Cursor, and any SKILL.md-based agent
