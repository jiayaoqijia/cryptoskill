Get Started

Get started in seconds

Clone the repo and copy any skill into your agent's workspace.

$ git clone https://github.com/jiayaoqijia/cryptoskill.git
$ cp -r cryptoskill/skills/exchanges/binance-spot-api ~/agent/skills/

Works with OpenClaw, Claude Code, and any MCP-compatible agent
