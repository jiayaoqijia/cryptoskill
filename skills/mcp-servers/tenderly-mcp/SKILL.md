# Tenderly MCP Server

Official Tenderly MCP server for simulating, debugging, and tracing smart contract transactions on 70+ EVM networks. Provides 43 tools for Virtual TestNet management, transaction simulation, on-chain tracing, contract inspection, and gas analysis.

## Installation

```bash
# Add to Claude Code
claude mcp add tenderly --transport http https://mcp.tenderly.co/mcp
```

## Features

- **Transaction Simulation**: Simulate transactions before sending them on-chain
- **On-chain Tracing**: Debug and trace smart contract executions
- **Virtual TestNets**: Create and manage virtual test environments
- **Contract Inspection**: Inspect contract state, storage, and ABI
- **Gas Analysis**: Analyze gas usage and optimize transactions
- **70+ EVM Networks**: Support for Ethereum, Polygon, Arbitrum, Optimism, Base, and more
